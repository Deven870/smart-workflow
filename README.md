# 🚀 Hybrid AI Workflow Assistant

> A mobile-first AI automation platform that converts unstructured inputs (text, files, voice) into structured outputs like tasks, summaries, and emails using a hybrid AI system.

---

## 📌 Overview

Hybrid AI Workflow Assistant is an intelligent productivity application that transforms raw user inputs into actionable insights.

Users can input text, upload documents, or use voice input, and the system automatically:
- Summarizes content  
- Extracts actionable tasks  
- Generates structured outputs  
- Stores and manages workflows  

The system leverages a **Hybrid AI Architecture**, combining local models (Ollama) with cloud AI (OpenAI/Gemini) for optimal performance, cost, and reliability.

---

## ⚡ Key Features

✅ Multi-input support (Text, File, Image, Voice)  
✅ AI Summarization & Task Extraction  
✅ Email Generation & Workflow Planning  
✅ Hybrid AI (Ollama + OpenAI/Gemini)  
✅ Auto To-do List Creation  
✅ SQL-based Data Storage  
✅ Dashboard Analytics  
✅ Clean UI + Dark Mode
